# MyFix

MyFix is a simple Gradle-based Java project.

## Project layout

- `src/main/java` — application source code
- `src/test/java` — test sources
- `build.gradle.kts` — Gradle build configuration
- `settings.gradle.kts` — project name and settings

## Prerequisites

- A compatible JDK installed
- Bash or a similar shell for running the Gradle wrapper on macOS/Linux

## Build

```bash
./gradlew build
```

## Test

```bash
./gradlew test
```

## Notes

The repository currently contains the Gradle project scaffold and JUnit 5 test setup.
